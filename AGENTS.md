# AGENTS Instructions

## Project Goal
Enhance the ISYS573 Retail Sales Dashboard with business-focused analytics features.

## Coding Rules
- Keep the dashboard clean and simple.
- Do not remove existing functionality.
- Maintain compatibility with Streamlit.
- Ensure filters work together correctly.
- Keep charts readable and business-friendly.
- Use clear variable names and comments when needed.

## Feature Requirements
- Add customer segment filtering.
- KPI cards must update dynamically.
- Existing quarter filtering must still function.
- Charts should refresh based on filters.

## Testing Requirements
- Dashboard must run without errors.
- All filters should work simultaneously.
- Charts and KPI cards should render correctly.
